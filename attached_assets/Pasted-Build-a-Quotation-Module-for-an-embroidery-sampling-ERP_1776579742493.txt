Build a Quotation Module for an embroidery sampling ERP system.

This module must support:

Client requirement capture
Suggested design attachments
Custom charge entries
Estimated costing summary
GST calculation
Revision tracking
Client feedback logging
Quotation approval workflow
Conversion to Swatch order
Conversion to Style order
Quotation PDF generation

Follow structure exactly as defined below.
